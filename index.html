Notions de HTML et CSS

Bonjour et bienvenu sur mon premier site internet. Je suis en plein apprentissage de HTML5 et CSS3. J'en profite donc pour faire une note des différentes notions. Avec un peu de chance, vous pourrez aussi piocher quelques infos utiles, sait-on jamais !

Le HTML (HyperText Markup Langage) est un langage de description de données qui permet la création de sites web. On écrit un document HTML dans un éditeur de texte classique.
