HTML pour les nuls !

Introduction

Le HTML (HyperText Markup Langage) est un langage de description de données qui permet la création de sites web. Il permet de présenter les données de type texte, image ou vidéos et il est structuré par des balises.

Une balise est présentée entre chevrons : 

  <balise>
Il existe deux formes de balises :
  • la forme complète qui s'écrit 
      <balise> du texte et d'autres balises imbriquées (les "enfants") </balise>
  • la forme auto-fermante qui s'écrit 
      <balise/>

Structure de base

La structure minimale d'un document HTML est de la forme suivante : 

 
    
      <! doctype html>                           <!--cette ligne précise quel langage HTML est utilisé (HTML5 ici)--!>
	  
      <html>                                     <!--pour commencer le document--!>
	  
        <head>                                   <!--l'en-tête du document--!>
          <meta charset="utf-8" />               <!--pour choisir le code d encodage UTF8--!>
          <title> Un titre </title>        <!--il faut absolument un titre !--!>
        </head>
		
        <body>                                   <!--et le corps du document--!>

        </body>
	  
     </html>

Le corps du document

Pour structurer la page web on peut ajouter entre les balises <body> et </body>:

  • des titres
  • des paragraphes de texte
  • des balises pour mettre en évidence certains mots (ou phrases) importants
  • des listes

Voyons tout de suite un exemple pour les notations :
