Voici mon premier site internet. Je suis en plein apprentissage de HTML5 et CSS3. J'en profite donc pour faire une note des différentes notions. Avec un peu de chance, vous pourrez aussi piocher quelques infos utiles, sait-on jamais !

Le HTML (HyperText Markup Langage) est un langage de description de données qui permet la création de sites web. On écrit un document HTML dans un éditeur de texte classique. Il permet de présenter de manière sémantique les données de type texte, image ou vidéos et il est structuré par des balises. Pour aller sur la page dédiée au HTML vous pouvez cliquer ici

Le CSS3 (Cascading Style Sheets) est un outil pour la mise en forme de votre site web qui vient agrémenter le contenu sémantique introduit grâce au HTML. On travaille la mise en page grâce à une imbrication de "boites" qui contiennent les différents éléments à organiser. Vous trouverez la page dédiée ici
