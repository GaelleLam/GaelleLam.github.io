Notions de HTML et CSS

Introduction

Le HTML (HyperText Markup Langage) est un langage de description de données qui permet la création de sites web. On écrit un document HTML dans un éditeur de texte classique. Il permet de présenter de manière sémantique les données de type texte, image ou vidéos et il est structuré par des balises.

Une balise est présentée entre chevrons : 

  <balise>
Il existe deux formes de balises :
  • la forme complète qui s'écrit 
      <balise> du texte et d'autres balises imbriquées (les "enfants") </balise>
  • la forme auto-fermante qui s'écrit 
      <balise/>

Structure de base

La structure minimale d'un document HTML est de la forme suivante : 

 
    
      <! doctype html>                           <!--cette ligne précise quel langage HTML est utilisé (HTML5 ici)--!>
	  
      <html>                                     <!--pour commencer le document--!>
	  
        <head>                                   <!--l'en-tête du document--!>
          <meta charset="utf-8" />               <!--pour choisir le code d encodage UTF8--!>
          <title> Un titre </title>        <!--il faut absolument un titre !--!>
        </head>
		
        <body>                                   <!--et le corps du document--!>

        </body>
	  
     </html>

Le corps du document

Pour structurer la page web on peut ajouter entre les balises <body> et </body> plusieurs éléments :

  • des titres
  • des paragraphes de texte
  • la mise en valeur du texte
  • des listes

Les titres

Les titres doivent être écrits entre les balises de type <h n> où n est un chiffre compris entre 1 et 6 inclus. Ce chiffre indique le niveau d'imbrication du titre. On distingue ainsi :

Les titres de niveau 1

qui doivent être codés dans notre fichier HTML sous la forme <h1> Titre de niveau 1 </h1>

Les titres de niveau 2

qui doivent être codés dans notre fichier HTML sous la forme <h2> Titre de niveau 2 </h2>

Les titres de niveau 3

qui doivent être codés dans notre fichier HTML sous la forme <h3> Titre de niveau 3 </h3>

Les titres de niveau 4

qui doivent être codés dans notre fichier HTML sous la forme <h4> Titre de niveau 4 </h4>
Les titres de niveau 5
qui doivent être codés dans notre fichier HTML sous la forme <h5> Titre de niveau 5 </h5>
Les titres de niveau 6
qui doivent être codés dans notre fichier HTML sous la forme <h6> Titre de niveau 6 </h6>
Il peut y avoir plusieurs titres de niveau inférieur après un titre de niveau supérieur. Par exemple, on peut insérer deux titres de niveau 2 suite à un titre de niveau 1. Par contre, il faut veiller à conserver l'ordre d'imbrication des titres et ne pas insérer un titre de niveau 2 avant un titre de niveau 1 par exemple.

Les paragraphes

Il est très facile d'insérer un paragraphe dans une page web en utilsant les balises <p> et </p> Par exemple on note : 

      <p> On écrit un premier paragraphe juste pour l'exemple même si on n'a pas grand chose à dire </p>
      <p> Et on rajoute un second paragraphe dans lequel on ne raconte toujours rien d'intéressant ! <p>
    et on obtient : 
	  
 On écrit un premier paragraphe juste pour l'exemple même si on n'a pas grand chose à dire 
 
      
 Et on rajoute un second paragraphe dans lequel on ne raconte toujours rien d'intéressant ! 
 
	

	
	
 La mise en valeur 

	
 Il se peut que vous ayez besoin de mettre en valeur certains mots, phrases ou paragraphes pour la compréhension sémantique de votre contenu.
    Dans ce cas vous pouvez utiliser trois balises <mark> </mark>, <em> </em> et <strong> </strong>. Voici un exemple d'utilisation : 
    
     code HTML : 
      <p> Voici un paragraphe dans lequel il y a des mots importants que l'on veut mettre en évidence : 
      on a d'abord un mot <mark> important </mark>, 
      puis un mot <em> très important </em> , 
      et enfin un mot  <strong> très très important ! </strong> 
      <p>
     résultat
      
 Voici un paragraphe dans lequel il y a des mots importants que l'on veut mettre en évidence : 
	  on a d'abord un mot  important ;, 
	  puis un mot  très important ; , 
	  et enfin un mot  très très important !  
	  

    

	



	
 Les listes 

	
 On distingue les listes ordonnées, les listes désordonnées et les listes de définitions.


	
 Les listes ordonnées permettent de présenter des éléments qui sont numérotés. Elles doivent être décrites entre les balises <ol> et </ol> 
    et chaque élément listé doit être indiqué entre les balises <li> et </li>.

	
 Voici un exemple : 
    
     
     code HTML : 
      <ol> une liste ordonnée
      <li> ligne 1 </li>
      <li> ligne 2 </li>
      <li> ligne 3 </li>
      </ol>
     résultat : 
      
     une liste ordonnée
	  
  1.  ligne 1 
    2. 
      
  2.  ligne 2 
    3. 
      
  3.  ligne 3 
    4. 
      

    

    


	
 Les listes désordonnées permettent de présenter des éléments avec des puces. Elles doivent être décrites entre les balises <ul> et </ul> 
    et chaque élément listé doit être indiqué entre les balises <li> et </li>.

	
 Voici un exemple : 
    
     
        code HTML : 
      <ul> une liste désordonnée
      <li> une ligne </li>
      <li> une autre ligne </li>
      <li> et encore une </li>
      </ul>
      résultat : 
      
     une liste désordonnée
	  
  •  une ligne 
    • 
      
  •  une autre ligne 
    • 
      
  •  et encore une 
    • 
      

    

    

	
	
 Les listes de définition, comme leur nom l'indique, permettent de présenter des termes et leur(s) définition(s). Elles doivent être décrites entre les balises <dl> et </dl> ,
    chaque terme listé doit être indiqué entre les balises <dt> et </dt> et les définitions entre les balises <dd> et </dd>. 
    Il peut y avoir plusieurs termes pour une définition et/ou plusieurs définitions pour un terme. 

	
 Voici un exemple : 
    
     
        code HTML : 
      <dl> une liste de définition
      <dt> un terme </dt>
      <dd> une definition </dd>
      <dd> et une autre definition pour le même terme </dd>
      <dt> un deuxieme terme </dt>
      <dd> et une deuxieme definition <dd>
      </dl>
      résultat : 
      
 une liste de définition
      
 un terme 

      
 une definition 

      
 et une autre definition pour le même terme 

      
 un deuxieme terme 

      
 et une deuxieme definition
