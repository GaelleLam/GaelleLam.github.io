HTML pour les nuls !

Introduction

Le HTML (HyperText Markup Langage) est un langage de description de données qui permet la création de sites web. Il permet de présenter les données de type texte, image ou vidéos et il est structuré par des balises.

Une balise est présentée entre chevrons : 

  <balise>
Il existe deux formes de balises : la forme complète qui s'écrit 
  <balise> du texte et d'autres balises imbriquées (les "enfants") </balise>
et la forme auto-fermante qui s'écrit 
  <balise/>

Structure de base

La structure minimale d'un document HTML est de la forme suivante : 

 
    
      <! doctype html>    <!--cette ligne précise quel langage HTML est utilisé (HTML5 ici)--!>
      <html>
        <head>

        <head>
      <body>

      <body>
     <html>
